# SPECTRA Challenge 

### Prerequisites

Before you begin, ensure you have the following prerequisites in place:

- Node 18.x
- Git 2.x

### Get  Started

1. Clone this repository to your local machine:

   ```
   git clone https://github.com/yschristian/SPECTRA-challenge.git
   ```

2. Challenge

   ```
   cd SPECTRA-challenge
   ```

3. run :

     ```
      npm run dev
     ```
